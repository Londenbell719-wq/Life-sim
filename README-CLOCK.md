# 🕐 Digital Clock with Multiple Time Zones

A beautiful, interactive digital clock that displays the current time in multiple time zones around the world.

## Features

✨ **Multiple Time Zones**
- Displays 20+ major world time zones
- Easy add/remove functionality
- Shows UTC offset for each zone

⏰ **Digital Display**
- Real-time digital clock
- Supports 12-hour and 24-hour formats
- Green glowing numbers with shadow effects

🔄 **Analog Clocks**
- Beautiful analog clock representation for each zone
- Hour, minute, and second hands
- Clock numbers positioned correctly

🎨 **Modern Design**
- Glassmorphism UI with frosted glass effect
- Smooth animations
- Responsive grid layout
- Blue gradient background
- Neon green accents

⚙️ **Controls**
- Switch between 12-hour and 24-hour formats
- Add new time zones dynamically
- Remove time zones with one click
- Default zones: UTC, EST, PST, JST

## Available Time Zones

- UTC (London/GMT)
- EST (New York)
- CST (Chicago)

- PST (Los Angeles)
- IST (New Delhi)
- JST (Tokyo)

- NZST (Auckland)

- CET (Berlin)
- EET (Cairo)
- MSK (Moscow)
- SGT (Singapore)
- HKT (Hong Kong)
- HST (Honolulu)
- AKST (Anchorage)
- BRT (São Paulo)
- And more!

## How to Use

1. **Open** `clock.html` in your web browser
2. **View** the current time in multiple zones
3. **Switch** between 12-hour and 24-hour format
4. **Add** new time zones from the dropdown
5. **Remove** any zone by clicking the × button

## Files

- `clock.html` - Main HTML file
- `clock-styles.css` - Styling and animations
- `clock-script.js` - JavaScript logic for time calculations
